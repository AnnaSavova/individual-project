The project

This app is a matching algorithm animator proposed by Sofiat Olaosebikan, my supervisor for this project, and developed as part of my final year individual project over the course of 6 months. The idea of this app was to expand on the work done on this project last year by Liam Lau, by adding Dr Manlove et al's Student-Project Allocation Algorith with Project Priority, as well as the version with the stability improvements introduced by Dr Iwama et al to the list of available algorithms for visualisation.

Although the visualisation does not work on canvas, the results of the SPA-P algorithms can be viewed via the console log under Developer Tools. An image of the intended visualisation can be viewed in the Evaluation Form.

This app was created by Liam for desktop use - so will look the best on larger screens - and I was unable to implement optimisation mobile devices due to time constraints so elements may not appear properly on smaller screens/windows.

To get started or learn more, head over to the Algorithms page where you can select an algorithm to run in different scenarios. Currently there are 5 options, 3 from the previous version, developed by Liam, and the 2 SPA-P implementations, developed by me, I have no current plans for adding more, especially not until the service can handle differences between algorithms in the number of agents
